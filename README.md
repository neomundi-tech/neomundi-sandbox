# ControlTowerAI - Sandbox

Interface interactive pour tester l'API ControlTowerAI de gouvernance thermodynamique des systemes IA.

## Ouvrir le Sandbox

**Option 1 — GitHub Pages** (zero install) :
Rendez-vous sur : [https://neomundi-tech.github.io/neomundi-sandbox](https://neomundi-tech.github.io/neomundi-sandbox)

**Option 2 — Local** :
Ouvrez `index.html` dans votre navigateur.

## Fonctionnalites

### 1. Gouvernance rapide
Envoyez des metriques LLM et observez la decision **ALLOW** (safe) ou **FLAG** (risque detecte).

### 2. Detection d'hallucinations
Testez si l'API distingue une reponse correcte d'une hallucination :
- Reponse correcte → **ALLOW**
- Reponse incorrecte → **FLAG** (hallucination detectee)

### 3. Metriques temps reel
Visualisez les compteurs live : requetes, decisions, G-score, latence.
